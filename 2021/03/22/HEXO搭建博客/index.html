使用 HEXO 搭建博客笔记

hexo
笔记
发布日期:   2021-03-22
更新日期:   2021-09-15
文章字数:   5.1k

前期准备工具

需要 node、npm（cnpm）、git、hexo 和 hexo 发布插件

  • 安装 hexo、hexo 发布插件命令如下

    cnpm i -g hexo-cli  # 安装hexo
cnpm install hexo-deployer-git --save  # 安装hexo发布插件

开始搭建

创建博客文件夹

hexo init hexoBlog

hexo 相关命令

hexo generate，简写成 hexo g

hexo s，使用本地端口预览

hexo clean // 清空

hexo -d 上传到github上（如果没有安装hexo发布插件将无法发布）

hexo g -d  // hexo -d 上传失败时可以使用这个

与 github 绑定

  • 需要在本地创建 ssh 密钥，然后在 github 上绑定该 ssh 密钥

    【ssh-keygen -t rsa】  此命令用于创建ssh密钥 可以在后面添加 【-C "备注"】 用于备注  
ssh密钥在 ~目录下的.ssh中的id_rsa.pub中（cd ~ 可以快速进入~目录）  
绑定后可以通过 ssh -T git@github.com 测试是否成功，成功时应该会提示  
Hi KeeGithub/linhieng.github.io! You've successfully authenticated, but GitHub does not provide shell access.

  • github上添加ssh密钥后需要在站点配置文件下修改url值和deploy值
    将url改为
    root: https://linhieng.com
    url如果设置不对那么会出现无法获取到css和js文件，因为路径错误
    deploy用于设置hexo如何提交

    # 注意冒号后面有空格, repo 的值就是仓库的 https 地址, branch 就是分支名

deploy:
  type: git
  repo: https://github.com/KeeGithub/linhieng.github.io.git
  # 如果配置了ssh后提交还需要添加账号和密码，那么就将【https://github.com/】改为 【】
  # git@github.com:KeeGithub/linhieng.github.io.git
  branch: master

  • 现在可以通过 hexo g -d 命令将hexo部署到github上面了，记得要先在public文件夹中添加 CNAME 文件

自选主题

下载主题

下载 该项目的 的master分支，将其整个文件放入Hexo文件的themes文件夹

使用主题

在站点配置文件下将 theme: landscape 改为 theme: hexo-theme-matery
hexo-theme-matery 就是刚才放入 themes 文件夹下的主题文件名
为了代码高亮好看一点，还需要在站点配置文件中修改开启 prismjs (代码块)

prismjs:
  enable: true

相关设置

添加自定义页面

在站点目录下的source下新建一个文件夹DIY, 该文件夹就是自己写的页面

需要在站点配置文件下修改 skip_render 的值为: DIY/**

  
skip_render: DIY/** # 渲染时跳过DIY目录，将DIY目录的内容直接复制到public中

在主题配置文件中的menu中添加DIY目录

  
menu:
  DIY页面: 
    url: /DIY

主题配置文件中的其他设置

  • 代码块折行，自行选择，若换行这代码块中不会出现横向滚动条

    break: false # 代码是否折行

  • 联系方式自定义修改

    socialLink:
  github:  https://github.com/keegithub
  qq: 790728740
  zhihu: https://www.zhihu.com/Linhieng

  • 显示文章字数

    安装插件
  npm i --save hexo-wordcount
在主题配置文件中开启
wordCount: true # 开启字数统计  
顺便开启更新日期
update: true # 更新日期

  • 开启 标签和分类 页

    先创建两个页面，可以通过下面代码快捷创建（创建后会在站点目录下的source文件夹中新增两个文件夹）
    hexo new page tags
    hexo new page categories
分别修改里面的md文件，各自添加如下内容
type: "categories" 
layout: "categories"

type: "tags"
layout: "tags"

然后在主题配置文件中的 menu 中开发 
  Tags:
    url: /tags
    icon: fas fa-tags
  Categories:
    url: /categories
    icon: fas fa-bookmark

tags: hexo # 文章标签，可以多个

雪花源代码

/*样式一*/
(function($){
	$.fn.snow = function(options){
	var $flake = $('<div id="snowbox" />').css({'position': 'absolute','z-index':'9999', 'top': '-50px'}).html('&#10052;'),
	documentHeight 	= $(document).height(),
	documentWidth	= $(document).width(),
	defaults = {
		minSize		: 10,
		maxSize		: 20,
		newOn		: 1000,
		flakeColor	: "#AFDAEF" /* 此处可以定义雪花颜色，若要白色可以改为#FFFFFF */
	},
	options	= $.extend({}, defaults, options);
	var interval= setInterval( function(){
	var startPositionLeft = Math.random() * documentWidth - 100,
	startOpacity = 0.5 + Math.random(),
	sizeFlake = options.minSize + Math.random() * options.maxSize,
	endPositionTop = documentHeight - 200,
	endPositionLeft = startPositionLeft - 500 + Math.random() * 500,
	durationFall = documentHeight * 10 + Math.random() * 5000;
	$flake.clone().appendTo('body').css({
		left: startPositionLeft,
		opacity: startOpacity,
		'font-size': sizeFlake,
		color: options.flakeColor
	}).animate({
		top: endPositionTop,
		left: endPositionLeft,
		opacity: 0.2
	},durationFall,'linear',function(){
		$(this).remove()
	});
	}, options.newOn);
    };
})(jQuery);
$(function(){
    $.fn.snow({ 
	    minSize: 5, /* 定义雪花最小尺寸 */
	    maxSize: 50,/* 定义雪花最大尺寸 */
	    newOn: 300  /* 定义密集程度，数字越小越密集 */
    });
});
