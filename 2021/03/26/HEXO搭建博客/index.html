使用 HEXO 搭建博客笔记

无标签
发布日期:   2021-03-26

前期准备工具

需要 node、npm（cnpm）、git、hexo 和 hexo 发布插件

  • 安装 hexo、hexo 发布插件命令如下

    cnpm i -g hexo-cli  # 安装hexo
cnpm install hexo-deployer-git --save  # 安装hexo发布插件

开始搭建

创建博客文件夹

hexo init hexoBlog

hexo 相关命令

hexo generate，简写成 hexo g

hexo s，使用本地端口预览

hexo clean

hexo

hexo -d 上传到github上（如果没有安装hexo发布插件将无法发布）

与 github 绑定

  • 需要在本地创建 ssh 密钥，然后在 github 上绑定该 ssh 密钥

    ssh-keygen -t rsa  # 创建ssh密钥 可以在后面添加 -C "备注" 用于备注  
ssh密钥在 ~目录下的.ssh中的id_rsa.pub中（cd ~ 可以快速进入~目录）  
绑定后可以通过 ssh -T git@github.com 测试是否成功，成功时应该会提示  
Hi KeeGithub/linhieng.github.io! You've successfully authenticated, but GitHub does not provide shell access.

  • github上添加ssh密钥后需要在站点配置文件下修改url值和deploy值
    将url改为
    root: https://linhieng.com
    url如果设置不对那么会出现无法获取到css和js文件，因为路径错误
    deploy用于设置hexo如何提交

    注意冒号后面有空格, repo 的值就是仓库的 https 地址, branch 就是分支名

deploy:
  type: git
  repo: https://github.com/KeeGithub/linhieng.github.io.git
  branch: master

  • 现在可以通过 hexo g -d 命令将hexo部署到github上面了，记得要先在public文件夹中添加 CNAME 文件

自选主题

下载主题

下载 该项目的 的master分支，将其整个文件放入Hexo文件的themes文件夹

使用主题

在站点配置文件下将 theme: landscape 改为 theme: hexo-theme-matery
hexo-theme-matery 就是刚才放入 themes 文件夹下的主题文件名
为了代码高亮好看一点，还需要在站点配置文件中修改开启 prismjs (代码块)

prismjs:
  enable: true

相关设置

添加自定义页面

在站点目录下的source下新建一个文件夹DIY, 该文件夹就是自己写的页面

需要在站点配置文件下修改 skip_render 的值为: DIY/**

skip_render: DIY/** # 渲染时跳过DIY目录，将DIY目录的内容直接复制到public中

在主题配置文件中的menu中添加DIY目录
menu:
  DIY页面: 
    url: /DIY

主题配置文件中的其他设置

  • 代码块折行，自行选择，若换行这代码块中不会出现横向滚动条

    break: false # 代码是否折行

  • 联系方式自定义修改

    socialLink:
  github:  https://github.com/keegithub
  qq: 790728740
  zhihu: https://www.zhihu.com/Linhieng

提搜索支持

安装  hexo-generator-search 插件
npm install hexo-generator-search --save
在站点配置文件中新增以下配置项
search:
  path: search.xml
  field: post

  • 关闭或开启的内容

    indexbtn: # 去掉主页的github按钮
  enable: false

music: # 关闭音乐
  enable: false

reward: # 关闭打赏
  enable: false

clicklove: # 关闭鼠标点击爱心
  enable: false

myProjects: # 关闭我的项目
  enable: false

mySkills: # 关闭我的技能
  enable: false

myGallery: # 关闭 在“关于”页面配置"我的相册"图片
  enable: false

githubLink: # 关闭 fork me
  enable: false

reprint: # 关闭付费转载
  enable: false

sharejs: # 关闭文章分享模块（通过QQ微信分享给好友之类的）
  enable: false

background: # 开启网站背景图
  enable: true

站点配置文件中的设置

  • 自定义 # Site 部分的内容

  • per_page 的值推荐设置为6的倍数

操作源代码

关闭动态切换背景颜色和修改背景颜色

.bg-color {
    background-image: linear-gradient(to right, #4cbf30 0%, #0f9d58 100%);
}

@-webkit-keyframes rainbow {
  /* 动态切换背景颜色. */
}

@keyframes rainbow {
    /* 动态切换背景颜色. */
}

.bg-color {background-image: linear-gradient(to right, #00d9ff 0%, #addff6 100%);}
